# Planilha Reconciliation v2.0

Sistema web para reconciliação mensal de dados de estagiários entre os setores financeiro e cadastro.

## Tecnologias
- Backend: Java 17, Spring Boot 3, PostgreSQL, JWT
- Frontend: React 18, TypeScript, Vite, TailwindCSS
- Infra: Docker, Nginx, Let's Encrypt

## Como executar
```bash
docker-compose up -d
